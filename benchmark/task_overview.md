# Benchmark Task Overview

## Task A — Inventory QA
Question answering over chemical and consumable state.

## Task B — Inventory Update Parsing
Parsing informal user reports into structured stock-changing events.

## Task C — Experiment Log Structuring
Converting fragmented messages and attachment descriptions into structured experiment memory.

## Task D — Weekly Project Summary
Generating evidence-grounded project summaries for PI review.

## Task E — Planning and Booking
Producing executable plans and conflict-free booking suggestions.

## Benchmark emphasis
ChemLabOps-Bench should evaluate:
- correctness
- faithfulness
- traceability
- ambiguity handling
- executability
