# Contributing

This project is currently in an early research-design stage.

## Good first contributions
- improve markdown documentation
- refine event schema
- create synthetic benchmark examples
- improve evaluation metric definitions
- propose cleaner repo structure
- add simple SQLite prototype code

## Contribution principles
- keep outputs traceable
- avoid overclaiming
- prefer realistic lab workflows
- keep the small-lab setting in scope
- preserve event-first design
