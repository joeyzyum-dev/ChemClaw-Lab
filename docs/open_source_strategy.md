# Open-Source Strategy

## Recommended repository shape

```text
chemclaw-lab/
├── README.md
├── PROJECT_STATUS.md
├── docs/
├── agents/
├── benchmark/
├── paper/
└── slides/
```

## What should be open-sourced first

1. project positioning and architecture
2. schema and event format
3. synthetic benchmark environment
4. agent prompt specifications
5. evaluation scripts
6. example case studies

## What can wait

- full production connectors
- institution-wide booking integration
- heavy UI work
- advanced vision modules

## Recommended release message

ChemClaw-Lab is an early-stage research-oriented starter framework for building messaging-native multi-agent assistants for chemistry laboratory operations.

## Open-source strengths of the project

- clear and realistic use case
- modular architecture
- benchmark-friendly design
- event-centric representation
- strong fit for AI-for-lab discussions
